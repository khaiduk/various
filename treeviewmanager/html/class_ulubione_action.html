Detailed Description

Klasa akcji menu Ulubionych.

Author:
Sławomir Gefert
Date:
2012-06-13

Klasa dziedziczy po klasie QAction. Używana w celu przechowywania dodatkowej informacji identyfikującej wybraną akcję. Sygnał trigerred() (kliknięcią na nią) jest przkazywany do jej slotu przeslijDalej(), który to pobiera jej "identyfikator" i wysyła sygnał z paramatrem, dzięki temu możliwa jest łatwa indetyfikacja, która to akcja została wybrana z menu.
